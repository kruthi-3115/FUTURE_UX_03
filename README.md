# CRM / Client Management Dashboard UI/UX

## 📌 Overview
This project is a CRM (Client Relationship Management) Dashboard UI/UX design created as part of the Future Intern UI/UX Design Internship. It is designed to help agencies manage leads, clients, tasks, and follow-ups through a clean and user-friendly dashboard.

## 🎯 Objective
- Manage leads effectively
- Organize client information
- Track lead pipeline
- Schedule tasks and follow-ups
- Improve team productivity

## 🛠️ Tool Used
- Figma

## ✨ Features
- Dashboard Overview
- Lead Pipeline
- Client Profile
- Tasks & Follow-ups
- Simple Navigation
- Clean UI Design

## 📂 Project Screens
- Dashboard
- Lead Pipeline
- Client Profile
- Tasks & Follow-ups

## 💼 Designed For
Digital Marketing Agencies and Small Business Teams

## 🚀 Outcome
This CRM dashboard provides an organized way to manage clients, monitor leads, track follow-ups, and improve workflow efficiency.
